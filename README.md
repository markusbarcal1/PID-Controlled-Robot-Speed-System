# PID-Controlled Robot Speed System



*Description:*



Built a real-time PID control system to maintain consistent motor speed across varying surface conditions using encoder feedback and interrupt-driven control.



*Key Features:*



* PID control implementation (P, I, D tuning)
* Real-time encoder-based speed estimation
* Interrupt-based control loop (timers + GPIO)
* PWM motor control via duty cycle adjustment



*Tech Used:*



* C (Embedded)
* FRDM-KL46Z
* MCUXpresso
* Motor driver (TB6612FNG)



*Results:*



* Maintained consistent motor speed within 3% error across multiple surfaces
* Demonstrated stable performance under varying load conditions
* Successfully implemented and tuned a full PID control loop in a real-time embedded environment



*What I learned:*



* How to design and tune feedback control systems
* Real-time data processing using interrupts
* Translating sensor data into system-level decisions
